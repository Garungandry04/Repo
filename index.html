Create Next Stack

The ultimate starter kit for Next.Js

last commit June 2024 license MIT npm v0.2.8

Create Next Stack is a website and CLI tool used to easily set up the boilerplate of new Next.Js apps.

Where Create Next App lets you choose a single template only, Create Next Stack lets you pick and choose an array of technologies often used alongside Next.js, freeing you of the pain of making them work together.
